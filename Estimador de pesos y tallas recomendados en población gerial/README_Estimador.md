## 🇬🇧 English

### What is this project about?
A spreadsheet-based tool designed to automate the auditing and monitoring of recommended weight and height metrics for a **geriatric population** dataset. Built to support operational control and data quality in healthcare or institutional settings.

### What does it do?
- Automatically estimates recommended weight and height ranges based on standard geriatric reference values
- Flags records that fall outside expected parameters
- Designed for non-technical users — no programming required, fully operational within Excel

### Tools
- **Microsoft Excel** — Advanced formulas, conditional formatting, data validation

### Use case
Useful for healthcare professionals, nursing homes, or any institution that needs to monitor and audit physical metrics in elderly populations without relying on specialized software.

## 🇵🇪 Español

### ¿De qué trata este proyecto?
Herramienta en hoja de cálculo diseñada para automatizar la auditoría y el seguimiento de métricas recomendadas de peso y talla en un dataset de **población geriátrica**. Orientada al control operativo y la calidad de datos en entornos de salud o institucionales.

### ¿Qué hace?
- Estima automáticamente los rangos recomendados de peso y talla según valores de referencia geriátrica estándar
- Identifica registros que caen fuera de los parámetros esperados
- Diseñada para usuarios no técnicos — no requiere programación, completamente operativa en Excel

### Herramientas
- **Microsoft Excel** — Fórmulas avanzadas, formato condicional, validación de datos

### Caso de uso
Útil para profesionales de salud, centros geriátricos o cualquier institución que necesite monitorear y auditar métricas físicas en población adulta mayor sin depender de software especializado.
